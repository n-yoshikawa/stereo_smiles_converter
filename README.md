# Stereo SMILES Converter
```
$ python encoder.py 
Supported case:
N[C@](O)(Br)C -> N[C@](O)(Br)C
N[C@@](O)(Br)C -> N[C@](C)(Br)O
N[C@H](O)C -> N[C@H](O)C
N[C@@H](O)C -> N[C@H](C)O
[C@H](N)(O)C -> [C@H](N)(O)C
[C@@H](N)(O)C -> [C@H](C)(O)N
[C@@](Br)(N)(O)C -> [C@](Br)(C)(O)N
Unsupported case:
[C@]1(Br)(Cl)CCCC(F)C1 -> [C@@]1(Br)(Cl)CC(F)CCC1
```
